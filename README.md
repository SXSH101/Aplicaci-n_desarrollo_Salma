# Aplicaci-n_desarrollo_Salma
Descripción del Proyecto

Esta aplicación fue desarrollada como parte de la materia de Cultura Digital. El objetivo del proyecto es crear un sistema que permita registrar información relacionada con mi desarrollo personal, académico y hábitos diarios, almacenando los datos en una base de datos PostgreSQL.
La aplicación cuenta con un frontend interactivo donde se ingresan los datos y un backend desarrollado en Python con Flask que se encarga de procesarlos y guardarlos en la base de datos.

Tecnologías Utilizadas:
- Python
- Flask
- PostgreSQL
- HTML
- CSS
- psycopg2

Funcionalidades Implementadas:
- Registro de usuario
- Registro de información académica
- Conexión a base de datos PostgreSQL
- Inserción de datos en múltiples tablas
- Interfaz gráfica amigable

Estructura del Proyecto
APLICACION_DE_DESARROLLO_PERSONAL/
app.py
requirements.txt
templates/
index.html
static/
style.css

Cómo ejecutar el proyecto
1. Instalar las dependencias necesarias.
2. Configurar la conexión a PostgreSQL en el archivo app.py (host, database, user, password y port).
3. Ejecutar el archivo app.py.
4. Abrir en el navegador la dirección http://127.0.0.1:5000

Objetivo Académico
Este proyecto permitió aplicar conocimientos sobre bases de datos, desarrollo backend y frontend, así como la conexión entre ambos mediante una aplicación construida con Flask.
